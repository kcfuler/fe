Core:

-   commands { cmd1:fn , cmd2:fn ...}
-   hooks { key1:fn[] , key2:fn[] ...}
-   应用元数据
-   prestes
-   plugins
-   ...

PluginAPI:

```ts
export default (api: IAPI) => {
api.registerCommand({
        key: '',
        fn(){}
    })
}

api.applyPlugins({
        key: '',
        initValue?: ''
    })
}
```

初始化阶段：按照一定的顺序依次获取所有 Prestes 和 Plugin
